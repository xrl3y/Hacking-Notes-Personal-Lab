¿Qué son los puertos?

Un puerto es un número lógico que acompaña a una dirección IP para identificar qué aplicación o servicio en un dispositivo está enviando o recibiendo datos. Piensa en la IP como la dirección de una casa y en el puerto como el número de habitación: la IP lleva los paquetes al dispositivo correcto y el puerto los entrega al programa correcto. Los puertos pueden usar TCP o UDP, y sirven para multiplexar muchas conexiones sobre una misma IP.

Los rangos de puertos más importantes:

- **0–1023 (Well-known)**: puertos estándar del sistema (requieren privilegios).
    
- **1024–49151 (Registered)**: puertos registrados para aplicaciones.
    
- **49152–65535 (Dynamic/Private)**: puertos efímeros usados por clientes.

# Puertos más conocidos y sus servicios (puerto — protocolo — servicio típico)

- **20 / 21 (TCP)** — FTP (datos / control) — transferencia de archivos.
    
- **22 (TCP)** — SSH — acceso remoto seguro y SFTP.
    
- **23 (TCP)** — Telnet — acceso remoto no seguro (obsoleto).
    
- **25 (TCP)** — SMTP — envío de correo electrónico.
    
###### - **53 (TCP/UDP)** — DNS — resolución de nombres.
    
###### - **67 / 68 (UDP)** — DHCP (servidor / cliente) — asignación automática de IP.
    
- **80 (TCP)** — HTTP — web no cifrada.
    
- **110 (TCP)** — POP3 — recepción de correo (antiguo).
    
- **119 (TCP)** — NNTP — grupos de noticias.
    
###### - **123 (UDP)** — NTP — sincronización de tiempo.
    
- **143 (TCP)** — IMAP — acceso/gestión de correo en servidor.
    
###### - **161 / 162 (UDP)** — SNMP (agent / trap) — monitorización de red.
    
- **194 (TCP)** — IRC — chat en tiempo real.
    
- **443 (TCP)** — HTTPS — web cifrada (TLS/SSL).
    
- **445 (TCP)** — SMB / Microsoft-DS — compartición de archivos e impresoras en Windows.
    
- **3306 (TCP)** — MySQL — base de datos MySQL/MariaDB.
    
- **3389 (TCP)** — RDP — Escritorio remoto de Windows.
    
###### - **5060 / 5061 (UDP/TCP)** — SIP / SIPS — señalización VoIP.
    
- **5432 (TCP)** — PostgreSQL — base de datos PostgreSQL.
    
- **5900 (TCP)** — VNC — control remoto de escritorio (VNC).
    
- **6379 (TCP)** — Redis — base de datos en memoria / caché.
    
- **8080 (TCP)** — HTTP alternativo / proxys / apps web (a menudo usado como puerto web alternativo).
